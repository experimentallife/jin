# Jin

## Overview

Platformed is a 2D game.
## Prerequisites

- [CMake](https://cmake.org) - 3.12 or higher
- [SFML](https://www.sfml-dev.org) - 2.4.2 or higher
## Usage

1. Clean project
```bash
$ make clean
```

2. Build project
```bash
$ make build
```

3. Run project
```bash
$ make run
```
